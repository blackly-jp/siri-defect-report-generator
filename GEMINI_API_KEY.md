# 🆓 How to Get a Free Google Gemini API Key

Google Gemini offers a generous **FREE tier** with no credit card required!

## Step-by-Step Guide

### 1. Go to Google AI Studio
Visit: https://makersuite.google.com/app/apikey

### 2. Sign in with Google Account
- Use any Google account (Gmail, etc.)
- No payment information needed

### 3. Create API Key
1. Click **"Create API Key"** button
2. Select **"Create API key in new project"** (or use existing project)
3. Copy your API key (starts with `AIza...`)

### 4. Use in the App
1. Go to https://siri-defect-report-generator.vercel.app/
2. Paste your API key in the "Google Gemini API Key" field
3. Click "Test API Key" to verify
4. Start using AI-powered generation!

## Free Tier Limits

✅ **60 requests per minute**
✅ **1,500 requests per day**
✅ **1 million tokens per month**

This is MORE than enough for defect report generation!

## Cost

- **Free tier**: Completely free, no credit card
- **Paid tier**: Only if you exceed free limits (very unlikely)

## API Key Format

- Starts with: `AIza...`
- Example: `AIzaSyD...` (32-39 characters)

## Security

- Your API key is stored locally in your browser
- Never shared with anyone except Google's API
- You can delete/regenerate it anytime at https://makersuite.google.com/app/apikey

## Troubleshooting

### "API Key Invalid"
- Make sure you copied the entire key
- Check for extra spaces
- Regenerate a new key if needed

### "Quota Exceeded"
- Wait a minute (60 requests/minute limit)
- Or wait until tomorrow (1,500/day limit)
